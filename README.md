Books Downloader
===
- Title:  `Books Downloader`
- Description: `This script parse and download books (English and Germany only) from https://library.fimsschools.com/all-pdf-books/`
- Authors:  `if164ever`
## Install & Dependence
- php

## Use
- download
  ```
  https://github.com/if164ever1/parserBooks.git
  ```
- for download books run in console
  ```
  php index.php
  ```
